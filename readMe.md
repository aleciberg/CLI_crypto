# CLI Crypto Tool

#### v2.0

## Description

#### Simple command line app to query CoinMarketCap for CrypotCurrency Quotes based on coin symbol (BTC, DOGE, ETH, etc). Enter command "coin" from anywhere to generate prompt and then enter symbol.
